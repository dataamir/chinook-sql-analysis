## Overview

End-to-end data analytics project using the Brazilian E-Commerce Public Dataset by Olist.
Covers the full workflow from raw CSV files to business recommendations.

**What I built:**
- Data cleaning and joining pipeline across 9 relational CSV files
- Exploratory data analysis with 9 visualisations
- RFM customer segmentation model
- Business insights with a 30/60/90 day action roadmap

## Key Findings

1. **Late deliveries tank review scores** — 2.4 stars vs 4.5 stars for on-time orders
2. **60%+ of customers only buy once** — serious retention gap
3. **Champions segment (11.5% of customers) drives 31% of revenue**
4. **Peak buying hours: 2–4 PM and 9–11 PM on weekdays**
5. **Electronics and Furniture have the lowest satisfaction scores** despite being highest value
